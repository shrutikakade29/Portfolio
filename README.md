# Portfolio - Shruti Kakade

Professional software developer portfolio showcasing modern web technologies and 3D visualizations.

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Framer Motion
- **Backend**: FastAPI (Python)
- **Styling**: Vanilla CSS

## 🚀 Setup
1. Clone the repo
2. Run `npm install` and `npm run dev` for frontend
3. Run `pip install -r requirements.txt` and `python main.py` in `backend/` for backend
